"Основи веб-дизайну"

З чого розпочати?

Усі веб-сайти будуються з допомогою мови розмітки HTML та каскадних таблиць стилів CSS. Обидві мови об’єднуються в професію
веб-верстальщика - людини, яка з допомогою HTML та CSS конвертує картинку з Photoshop у сторінку, яку розуміє веб-браузер та вміє правильно відобразити користувачу. Для динамічності веб-сайту використовується мова програмування JavaScript.

HTML

HTML (Hypertext Markup Language) — це не мова програмування, це мова розмітки, яка каже вашому браузеру, як відображати вміст веб-сторінки.

Детальніше...
CSS

Cascading Stylesheets (CSS, каскадні таблиці стилів) — це код, що використовується для стилізації сайту(наприклад: колір тексту, розташування, позиціонування елементів).

Детальніше...
JavaScript

JavaScript — це мова програмування, яка приносить інтерактивність до вашого веб-сайту (наприклад: реакція на натиски кнопок, динамічна зміна стилів, анімація).
